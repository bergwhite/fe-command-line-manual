## 在Python3中使用爬虫框架Scrapy

```

pip install pypiwin32  // 安装Python的win32库
pip install scrapy  // 安装爬虫框架Scrapy
scrapy runspider test.py -o bw2.json --set=FEED_EXPORT_ENCODING=utf-8  // 启动命令
```
